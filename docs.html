• Export

Documentation

The Events Export application queries the Watchwith Airings API using the given parameters, and generates a tab-delimited file (TSV) representing all events in the query response. Exported files can be imported into a business intelligence tool to support analysis of event-level metadata.

Query parameters:

  • Channel: The Watchwith channel ID for the channel on which Airings are to be queried.
  • Access Key: The Watchwith Application Key to which the Airings to be queried are syndicated.
  • Start Date / End Date: The date range for the query. Specified date is from 00:00:00-23:59:59 in the channel's time zone.

Actions:

  • Export TSV: Runs the query with given parameters, and generates a TSV file with the results. File name is formated as such: export-{channel}_{key}_{date}_{utc time}.tsv
  • Copy URL to Clipboard: Copies configured URL to the system clipboard. Useful to copy specific query URLs after they have been run.

Information:

  • Columns: Columnar display of events from last export during current session.
  • Export Log: List of previously run exports during current session.
